# Spring Core - Exercise 4

## 🎯 Objective
Set up a Maven project for a Library Management application with required Spring dependencies and Java 1.8 compiler plugin.

---

## 🛠️ Tasks

1. **Create a Maven project** named `LibraryManagement4`
2. **Add dependencies** in `pom.xml`:
   - `spring-context`
   - `spring-aop`
   - `spring-webmvc`
3. **Configure Maven Compiler Plugin** for Java 1.8

---

## 📁 Files Included

- `LibraryManagement4/` — Complete Maven project with configuration
- `pom.xml` — Includes required dependencies and Maven plugin
- `Build Success Screenshort.png` — Screenshot of successful Maven build
- `Exercise 4.docx` — Detailed documentation of the setup steps and configuration used

---

- No Java source files were required for this exercise.
